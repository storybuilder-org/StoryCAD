You can use a wide range of input devices with StoryCAD:  keyboard, virtual keyboard, pen or stylus, mouse, and touch, depending on your computer or tablet â€˜s capabilities.
Invoke pop-up menus using the following actions:

Input		Action
Mouse		Right Click
Keyboard		Menu Key (see below)
Touch		Long Press on item
Pen		Barrel Button Press or Long Press on item
Touchpad		Right Touchpad button
