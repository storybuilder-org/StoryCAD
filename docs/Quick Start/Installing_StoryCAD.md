StoryCAD is available for Windows and macOS. Follow the installation instructions for your platform below.

### Download StoryCAD

Visit [storybuilder.org](https://storybuilder.org) to download StoryCAD. The website has buttons linking to all supported app stores for your platform.

### Windows Installation

**System Requirements:**
- Windows 10 (version 19041.0 or higher) or Windows 11
- x64 or ARM64 processor
- Internet connection for installation

**First Launch:**
After installation, find StoryCAD in your Start Menu and launch it.

### macOS Installation

**System Requirements:**
- macOS 10.15 (Catalina) or later
- Apple Silicon (M1 or newer)
- Internet connection for installation

**First Launch:**
After installation, find StoryCAD in your Applications folder or Launchpad.

### Platform Differences

Windows and macOS versions have identical outlining functionality. See [Platform Differences](../Front%20Matter/Platform_Differences.html) for details on text editing differences between platforms.

### Getting Started

After installation, see [Starting and Stopping StoryCAD](Starting_and_Stopping_StoryCAD.html) for information on launching the application and [Reading and Writing Outlines](Reading_and_Writing_Outlines.html) for creating your first story outline.
